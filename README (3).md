## 🚀 Deploy in 3 Steps

1. Upload both files to your GitHub repo
2. Enable GitHub Pages → Settings → Pages → main branch → / (root)
3. Your URLs:
   - Dashboard: `https://YOUR-USERNAME.github.io/REPO/dashboard.html`
   - Quiz: `https://YOUR-USERNAME.github.io/REPO/quiz.html` (auto-configured per session)

## ✅ How to Use for Each Class

### Before the class:
1. Open `dashboard.html` → Login
2. Click **➕ New Session**
3. Enter: College Name, Date, Class Name, Discipline
4. Click **Create Session** → Copy the quiz link

### During the class:
1. Share the quiz link via WhatsApp / Zoom chat / projector
2. Keep `dashboard.html` open → Go to **📋 All Sessions** → Click **Live Leaderboard**
3. Watch student scores appear in real time!

### After the class:
1. Click **⬇️ Download CSV** on that session
2. CSV has: College, Date, Class, Student Name, Dept, Roll No, Score, Time
3. Use for **CAS / NAAC / Audit / Annual Report** evidence
